About iOS JSON Benchmark

A simple testbed to check several JSON parser performances on iOS. Supported JSON parsers

  • JSONKit / Objective-C
  • TouchJSON / Objective-C
  • NextiveJSON / Objective-c
  • SBJSON / Objective-C
  • NSJSONSerialization (Apple) / Objective-C
  • RapidJson /C++
  • Gason /C++ Other parsers #####Parser and JSON file options
